## ROLE CAPABILITIES

STUDENT CAN:

• View quizzes created by faculty
• Attempt quizzes
• See correct answers after submission
• View past attempts
• See score and timeline
• See quizzes appear automatically (poll every 10 seconds)
• Logout

STUDENT CANNOT:

• Create quizzes
• Access faculty routes
• View other students' attempts

FACULTY CAN:

• Create quizzes
• Add multiple questions
• Add multiple options per question
• View previously created quizzes
• View attempts ONLY for their own quizzes
• View student name
• View student email
• View score
• View attempt timeline
• View attempts sorted by newest first
• Logout

FACULTY CANNOT:

• Attempt quizzes as student
• View quizzes created by other faculty
